# AI Project Context

## Vision
Scroll Narrative Framework (SNF) is an AI-native, renderer-agnostic engine for building narrative web experiences.

## Mission
Model narrative first, implementation second.

## Core Principles
- Architecture First
- Narrative Driven Development (NDD)
- Data-Oriented Design
- Renderer Agnostic
- Accessibility by Default
- Performance by Default

## AI Rules
- Think in Intent, Narrative, Sequence, Cue, Behavior, Progress.
- Never design around a specific animation library.
- Treat React, GSAP, CSS Scroll Timeline, WAAPI, Lottie and Three.js as adapters.
- Preserve package boundaries.
- Never bypass architectural specifications.

## Long-term Goal
Generate complete narrative experiences from natural language and compile them into runtime-specific implementations.
