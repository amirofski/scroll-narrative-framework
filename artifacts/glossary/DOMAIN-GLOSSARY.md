# Domain Glossary

## Experience
The complete interactive narrative presented to the user.

## Narrative
An ordered collection of scenes.

## Scene
A bounded narrative segment with a clear emotional goal.

## Cue
The smallest executable narrative instruction bound to progress.

## Progress
A normalized value (0..1) representing advancement through an experience.

## Behavior
A runtime action executed in response to cue evaluation.

## Layer
An independently rendered visual plane within a scene.
