# Scroll Narrative Framework Engineering Handbook

Status: Draft

This document defines the engineering standards for the Scroll Narrative Framework (SNF).

## Principles
- Architecture First
- Data-Oriented Design
- Narrative Driven Development (NDD)
- Testability
- Performance by default
- Accessibility by default
- AI-native repository

## Definition of Done
- Architecture reviewed
- Tests added
- Documentation updated
- Public API reviewed
- Examples included
- Performance budget respected
- Accessibility considered
